# Personal Dashboard

A simple and elegant dashboard that displays a live clock, weather information, cryptocurrency prices, and a beautiful background image every time the page loads.

## Features

* Live digital clock
* Random background images from Unsplash
* Real-time Bitcoin price data from CoinGecko
* Current weather based on user's location
* Geolocation support
* Clean and responsive UI

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Unsplash API
* CoinGecko API
* OpenWeather API (via Scrimba proxy)

## Project Structure


## Installation

1. Clone the repository

```bash
git clone https://github.com/sahilmishraaa/chrome-extension-personal-dashboard.git
```

2. Open the project folder

```bash
cd chrome-extension-personal-dashboard
```

3. Open `index.html` in your browser

or

Load it as a Chrome Extension:

* Open Chrome
* Navigate to `chrome://extensions`
* Enable **Developer Mode**
* Click **Load unpacked**
* Select the project folder

## APIs Used

* Unsplash for random background images
* CoinGecko for cryptocurrency data
* OpenWeather for weather information
